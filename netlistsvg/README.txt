NetlistSvg.NET Distribution Package
====================================

This directory contains the distribution files for NetlistSvg.NET.

Contents:
---------
• NetlistSvg.1.0.0.nupkg
  The NuGet package containing the library. Install via:
  dotnet add package NetlistSvg
  or
  Install-Package NetlistSvg

• README.md
  Main project documentation with installation instructions, quick start
  examples, and API reference.

• AI_DOCUMENTATION.md
  Comprehensive documentation optimized for AI agents, including detailed
  API reference, usage patterns, and best practices.

• LICENSE
  MIT License for NetlistSvg.NET. Note that the package includes third-party
  components with their own licenses (see THIRD_PARTY_LICENSES.txt in the
  package or call LicenseInfo.GetThirdPartyLicenses()).

Quick Start:
------------
1. Install the package:
   dotnet add package NetlistSvg

2. Or install from this directory:
   dotnet add package NetlistSvg.1.0.0.nupkg

3. See README.md for usage examples
